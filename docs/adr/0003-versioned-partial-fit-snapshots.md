# 逐模型发布版本化拟合快照

一个拟合批次中的模型相互独立，免费、观察、未知或无效模型不应阻止其他有效模型更新。系统先不可变保存整个拟合提交，再逐模型生成处理结果；同批中通过校验并允许自动生效的候选以一次原子覆盖合并生成新的完整快照，未提交或未通过的模型保留原生效值。每个快照记录各模型自己的来源批次、数据窗口和版本水位，不把混合来源伪装成同一批数据。

## Consequences

接口返回逐模型状态而不是简单的整批成功或失败。相同提交标识与相同内容保持幂等，标识相同但内容不同产生冲突；旧来源水位不能覆盖新值。回滚通过创建一个引用历史拟合的新快照完成，不倒退版本水位或修改历史提交。
