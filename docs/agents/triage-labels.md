# Triage labels

| 标准角色 | GitHub标签 | 含义 |
|---|---|---|
| `needs-triage` | `needs-triage` | 等待维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，代理可独立实现 |
| `ready-for-human` | `ready-for-human` | 需要人工实现或决策 |
| `wontfix` | `wontfix` | 不会处理 |

技能提及标准角色时，必须使用本表对应的GitHub标签。需要调整词汇时修改右侧GitHub标签列，并同步仓库实际标签，避免创建语义重复的标签。
